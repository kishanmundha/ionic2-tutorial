# Ionic 2 Tutorial

Ionic 2 tutorial design in Ionic2 framework to begainner.

## Locally run
---

### Clone repository

```sh
$ git clone https://github.com/kishanmundha/ionic2-tutorial.git
cd ionic2-tutorial
```

### Install npm package

```sh
$ npm install
```

### Serve app

```sh
$ npm start
```
